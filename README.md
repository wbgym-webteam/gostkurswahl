# "Kurswahlsystem" / Course selection system

## Introduction

This is a full stack web application for students to select their courses for their Abitur at the Weinberg Secondary School Kleinmachnow.

## Techstack

|Section|Language/Framework|
|:-----:|:----------------:|
|Frontend|vue.js|
|Backend|Python Flask|
|DB|SQLite|
